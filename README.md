> ⚠️ This project contains anonymized data and screenshots to protect confidential business information.

---

# 📌 Background

The company previously conducted an **"Assertive Behavior for Sustainability Improvement"** bootcamp.

One of its key outcomes was a monthly evaluation program designed to measure assertive behaviors across both individuals and divisions.

Initially, the evaluations were conducted using Google Forms. Over time, however, the process became difficult to sustain because employees had to evaluate too many colleagues, score calculations were performed manually, and administrators struggled to monitor participation.

As participation declined, divisional evaluations eventually stopped, putting the organization's culture improvement initiative at risk.

---

# 💡 Solution

I redesigned the entire evaluation workflow and developed an internal web application that transformed a manual process into an automated assessment system.

Instead of asking employees to evaluate everyone, each employee now evaluates only **four randomly assigned colleagues**, while every employee still receives **four independent evaluations** to ensure fairness.

The application also automates score calculation, generates rankings, and provides administrators with real-time submission monitoring.

---

# ✨ Key Features

### 📝 Guided Assessment Workflow

A simple step-by-step evaluation experience that reduces complexity for employees.

### 🎲 Automatic Peer Assignment

Each employee is automatically assigned four colleagues to evaluate, ensuring balanced participation while minimizing evaluation fatigue.

### 🏆 Individual & Division Rankings

Automatically generates monthly rankings for both employees and divisions without manual calculations.

### ⚡ Automatic Score Calculation

Eliminates manual recaps by calculating evaluation scores automatically.

### 📊 Submission Monitoring

Administrators can instantly monitor submission progress and identify employees who have not yet completed their evaluations.
